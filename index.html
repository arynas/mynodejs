Simple Web with NodeJS

Nama Sekolah/Instansi Jurusan Angkatan Motivasi Mengikuti Workshop
Moch. Ari Nasichuddin UGM Departemen Teknik Elektro dan Teknologi Informasi 2016 Node.js dalam perkembangannya mengalami pertumbuhan pesat menyaingi teknologi yang lain. Salah satu kelebihan Node.js adalah adanya single-threaded dan menggunakan concurrency model yang berbasiskan event loop. Ini berarti Node.js dapat menangani operasi secara bersamaan tanpa mesti menunggu satu persatu. Sehingga memungkin program berjalan lebih cepat. Dengan keunggulan seperti itu maka Node.js mempunyai potensi diadopsi di beberapa project yang menginginkan program berjalan lebih cepat. Keikutsertaan saya dalam workshop ini selain untuk belajar, juga ingin berbagi ke kawan yang lain terkait Node.js. Harapannya akan ada sharing knowledge tentang Node.js dan Javascript pada umumnya.
